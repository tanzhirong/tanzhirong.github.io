Part 1

In a bike sharing system the process of obtaining membership, rental, and bike return is automated via a network of kiosk locations throughout a city. I will try to combine historical usage patterns with weather data to forecast bike rental demand in the Capital Bikeshare program in Washington, D.C. Predictions are evaluated using the root mean squared logarithmic error (RMSLE).

For the purpose of evaluating the models, I divide the training set into two parts: first 15 days of each month as the new training set and 16th to 19th as the validate set. Using this new training set, I will then fit a linear model on count numbers against (a subset of) the time and weather variables.

First, we note that the “Year” and “Day” variables are not required. I choose to select the “season” variable instead of the “month” variable (note both are highly correlated). In addition, since the temperature felt will affect riding rate more than the actual temperature, I will include “atemp” but exclude “temp”.

Next, we will transform the Season, Hour, Weather as factor variables.

Part 4

Using the methods from the previous 3 parts, I will create a final nonparametric model, and use it to predict my test set.

#Since we have to replicate the process for both train and validation set, 
#and the overall training and test set, I'll write some functions.

daycountfitted = function(dataset) {
  daymean <- aggregate(dataset$new.count, by=list(dataset$daylabel), mean)
  colnames(daymean) = c("daylabel","logcount")

  #Find the OPTIMAL bandwidth for means against daylabel.
  gridmat.d = matrix(0, ncol=2, nrow=100)
  gridmat.d[, 2] = (seq(20, 200, length=100))
  gcvs3 = gcvplot(logcount~daylabel, data=daymean, alpha=gridmat.d, kern="gauss",ev=dat())
  optband = max(gcvs3$alpha[gcvs3$values == min(gcvs3$values),2])
  optband

  #Fit the local linear regression model.
  locfitopt3 = locfit(logcount~daylabel, data=daymean, 
                      alpha=c(0,optband), deg=1, maxk = 1000)
  return (locfitopt3)
}

predict.bikes = function(training, testing) {
  locfitdaylabel = daycountfitted(training)
  training$new.daylabel <- predict(locfitdaylabel, newdata=training$daylabel)
  testing$new.daylabel <- predict(locfitdaylabel, newdata=testing$daylabel)
  
  gamfit.5d <-gam(new.count ~ s(new.daylabel) + s(temp) + s(atemp) + s(humidity) + s(windspeed)
             + s(atemp*workingday)
             + hour + season + workingday + weather,
            data=training)
  predict.5d = predict(gamfit.5d, newdata = testing)
  normal.count = exp(predict.5d) -1
  return (normal.count)
}
