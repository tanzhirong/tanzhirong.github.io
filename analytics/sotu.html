USA State of the Union Analysis¶

Data Processing¶

First, we write Python code that parses each SOU address, finding end-of-sentence markers.

In [1]:
import json  # in import cell
import numpy as np
import matplotlib.pyplot as plt
import nltk
from nltk import tokenize
#nltk.download('punkt')
from nltk.tokenize import sent_tokenize, word_tokenize
import statistics as stat

with open('speeches.json', 'r') as f:
    speeches = json.loads(f.read())

for i in range(len(speeches)):
    speeches[i]['sentence'] = sent_tokenize(speeches[i]['text'])
    for sentence_i in range(len(speeches[i]['sentence'])):
        speeches[i]['sentence'][sentence_i] = speeches[i]['sentence'][sentence_i].rstrip("\n\r")
    #Create template of dimension (#sentences)
    speeches[i]['word'] = np.empty([len(speeches[i]['sentence'])])
    for j in range(len(speeches[i]['sentence'])):
        #Find number of words
        speeches[i]['word'][j] = len(word_tokenize(speeches[i]['sentence'][j]))

Here, we use the function sent_tokenize and word_tokenize from the nltk.tokenize package to split the SOU addresses into sentences. Note that the examples of "Mr." are specifically handled in this package, i.e. the sentence will not split there.

Regression¶

How has the length of the address changed over the 200 years? For each year, we compute the number of sentences in the address, and the mean sentence length in words for that year.

Recall that $\hat{Y} = X\hat{\beta}$, where $\hat{\beta}$ is given by $(X^T X)^{-1} X^T Y$. Hence $\hat{Y} = X (X^T X)^{-1} X^T Y$.

Note that from the regression, the coefficient for year is 0.785. This implies that the expected increase in number of sentences for every passing year is 0.785.

However, from the points, notice that while the trend is increasing overall, there is a structural change in the year ~1910, when the number of sentences decrease before increasing again, although at a slower pace than before 1910. The structural change will be explained in Part (c).

Next, we analyze the regression of average length of sentence against year. Here, the coefficient for year is -0.123, implying that for every passing year, the average number of words in a sentence decreases by 0.123, hence each sentence is getting shorter and shorter.

Overall in general, we observe that as the years pass, the number of sentences tends to increase, but the average length of the sentences decreases. This could be due to changes in linguistic preferences - people in the past tend to contain more of their words in a sentence, hence as the sentence length decreases, the number of sentence must increase to convey the same (or more) amount of content. There is a structural change in ~1910s which will be further explored in the next part.

In this case, note that the total number of words is the best indication of the total length of the speech compared to either the number of sentences and average number of words in sentence. Here, we are measuring the overall length of speeches have changed.

The coefficients for the first set of years is 152, and the 2nd set of years is 8.6. This implies that before 1913, the expected increase in speech length for every passing year is 152, and after 1913, this number drops to 8.6. Hence, the speech grows at a much faster pace before 1913 compared to after 1913. This is because before 1913, the President primarily only submitted a written report to Congress. After 1913, Woodrow Wilson, the 28th U.S. President, began the regular practice of delivering the address to Congress in person.

Hence, since the speech was only done in writing before 1913, the increase in length is more drastic, since the President picks up the habit of writing the report to convey more of his objectives, and it is easier to say more in a report than in person. After 1913, when the speech is done in person, there is a drastic drop, since it is not possible to give too long a speech - this also accounts for why the increase in speech length after 1913 is only 8.6 which is close to 0 - it is not possible for the speech to grow too long given that the timing is rather fixed.
